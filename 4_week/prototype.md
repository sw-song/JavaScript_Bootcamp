# 자바스크립트 객체에 관하여
---

## Class
- ES6부터 class 문법을 제공하지만 파이썬과 같이 다른 언어에서 사용하는 class와는 차이가 있다.
- 따라서 자바스크립트는 class가 제공되긴 하나 class기반 언어는 아니다.
- 자바스크립트는 prototype(프로토타입) 기반 언어다.

## Object
- 객체가 필요한 기본적인 속성들을 가지고 있는 전역 객체(최상위 객체)

## __proto__
- 빈 객체 하나를 생성하게 되면, 객체 안에 __proto__라는 속성이 포함된 것을 볼 수 있다.
- __proto__는 상위 객체를 가리키며, 정확히는 해당 객체의 prototype(Object Prototype)을 참조한다.

## prototype
- prototype은 생성자 함수만 가질 수 있는 객체의 '원본'을 뜻한다.
- new를 통해 객체를 생성하는 경우 생성자 함수를 사용한 것이므로 생성자(constructor)는 자기 자신이다.
